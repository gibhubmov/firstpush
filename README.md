# Futuristic Web App UI

A modern, futuristic single-page website with glassmorphism effects, smooth animations, and responsive design.

## Features

- **Dark Theme**: Sleek dark background with gradient overlays
- **Glassmorphism**: Blur effects, transparency, and soft shadows
- **Smooth Animations**: CSS transitions and JavaScript-powered effects
- **Responsive Design**: Optimized for both desktop and mobile devices
- **Interactive Elements**: Hover effects, scroll animations, and micro-interactions
- **Modern Typography**: Using Inter font for clean, readable text

## Sections

1. **Navigation Bar**: Sticky navbar with smooth hover animations
2. **Hero Section**: Large heading, subtitle, CTA button, and animated background shapes
3. **Features Section**: 4 feature cards with icons, hover effects, and scale animations
4. **Stats Section**: Animated counters that trigger on scroll
5. **Footer**: Clean footer with social links and company info

## Technologies Used

- HTML5
- CSS3 (with modern features like backdrop-filter, CSS Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## Design Highlights

- **Color Palette**: Dark backgrounds with purple/blue gradients and neon accents
- **Typography**: Clean, modern font with gradient text effects
- **Animations**: Smooth transitions, floating shapes, counter animations
- **Effects**: Glassmorphism, glowing borders, parallax scrolling

## How to Run

1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. Enjoy the futuristic experience!

No build process or server required - it's a static website.

## Browser Support

- Chrome 76+
- Firefox 70+
- Safari 12+
- Edge 79+

*Note: Some effects may not work in older browsers due to CSS backdrop-filter support.*
